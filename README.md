# Simple Interest Calculator

This repository contains a Bash script that calculates **Simple Interest** based on user input.

## Formula

Simple Interest (SI) is calculated as:
SI = (Principal × Rate × Time) / 100

Where:

- **Principal**: The initial amount of money
- **Rate**: Interest rate per annum
- **Time**: Time period in years

## Usage

Run the script in a terminal:

```bash
bash simple-interest.sh
Enter the following when prompted:
```
Principal amount

Rate of interest

Time period (in years)

The script outputs the calculated simple interest.

Example
Enter Principal: 1000
Enter Rate of Interest: 5
Enter Time (years): 2
Simple Interest is: 100
